## 迭代文档模板

每个迭代文档应包含：
1. **背景** - 为什么做这次迭代
2. **目标** - 要达成什么
3. **决策** - 重大技术/产品决策
4. **变更** - 具体改了什么
5. **踩坑记录** - 遇到的问题和解决方案
6. **技术债务** - 留下的待办
7. **下一步** - 后续计划
