```

## 保留自 v1 的有效设计

- **即兴分析必须基于原文**（ADR-005/决策5）：article/paper/repo 选中时实时抓原文，
  失败降级摘要且显式标注——已在 ephemeral-chat.js 实现。
- **深度调研路径**（v1 §3b）：搜索模糊主题 → 查已有内容 → 不足触发公网调研 →
  确认后转常驻 Topic。M3 与选题/活页整合时再启用。
- **信源池**（v1 §6）：GitHub Trending / Reddit 等"内容池"不创建 Source 身份，
  内容直接进 Feed 标注池名——M2 接 GitHub Trending 时沿用。
```
