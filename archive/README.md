## 📚 如何使用归档文档

### ✅ 适合查阅归档的场景

1. **了解历史决策** - 为什么选择某个技术方案？
2. **参考需求分析** - 初期的用户需求是什么？
3. **理解演进过程** - 架构是如何迭代的？

1. **开发参考** - 请使用根目录的最新文档
2. **架构设计** - 请使用 `ARCHITECTURE.md`
3. **API 设计** - 请使用 `MVP-HANDOFF.md`
