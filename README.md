# Knowledge Workbench · 知识工作台

> 一个人做内容、为自己边做边搭的工作台，在这里公开记录它长大的过程。**欢迎 ⭐ star 关注、来交流想法。**

**收藏了几百篇，真到动笔却一篇都想不起来？** 收藏不等于拥有。

它的信念很简单：**AI 吃掉一切不产生理解的杂活**——采集、翻译、摘要、聚类、排版、平台适配；**判断、观点、表达，留给人。** 于是你读到的，能真正变成你写出来的、发出去的，再让结果反哺下一次读什么、写什么——一条转起来的**内容飞轮**：

> **读 → 想明白 → 写出来 → 发出去 → 复盘反哺 → 更懂下一篇写什么**

> **飞轮还差最后半环**：「发布 → 数据 → 复盘 → 自进化」正在做，目前仍是手动 / 愿景——这也是本项目北极星（内容是否让读者更有收获）真正的落地抓手。

---

## 🎯 它解决什么问题

信息过载时代，读得多不等于懂得多。这个工作台把「信息 → 认知 → 内容」做成一条转起来的线：

```
多源信息流入（宽）
    ↓ 采集 + 筛选（AI 全自动）
主题弹药库（收窄：素材归入长期研究的主题）
    ↓ 探讨（人的主场：带着问题与 AI 对话，沉淀判断）
深稿（价值凝结点）
    ↓ 裂变（AI 全自动：长文 / thread / 口播 / 图文…）
多平台内容（放宽）
    ↓ 发布 + 复盘
反哺下一次（闭环：让选题与判断随每次复盘持续变好）
```

> 目前 **「多源流入 → 深稿」** 已跑通；**「发布 → 复盘 → 反哺」那半环仍在做**。

分工原则：**AI 吃掉一切不产生理解的环节**；**「决定写什么、核心论点、判断」留在人手上**。

## ✨ 有什么功能

### 📥 资讯 —— 多源采集 + 简报
- 多源自动同步：AI 精选榜、Hacker News、GitHub Trending，以及你登记的 X 账号 / 博客 / 播客 / B站 / YouTube / 公众号
- 每天一份简报：今日热点 + 按你关注的主题精选 + 选题建议；周报 / 月报看趋势与主题演进

### 📝 素材 —— 沉淀
- 读到有用的一键存为素材，自动起标题、自动归入相关主题
- 语义搜索：用大白话就能找回攒过的料，也能把 AI 当搜索引擎问整个素材库

### 📚 主题 —— 知识加工（核心资产）
- 每个主题是一篇 AI 帮你持续维护的综述：存进新素材，它自动更新认知、标出观点冲突、记下每次修改
- 随时和 AI 探讨这个主题（拿你自己的综述和素材当弹药）

### 💡 灵感 —— 选题种子
- 选题种子收集箱：刷到能写的、脑里冒出的、随手记的，先收下来，别忙完就忘
- 丢链接 / 会议纪要 / 音频 / PDF 进来，AI 帮你读懂、存成素材、可提为灵感
- 写作看板：帮你看清哪条"料够了、可以写"，决定先写哪一条

### ✍️ 创作 —— 内容生产
- 挑素材起稿，按平台生成初稿（长文 / thread / 口播 / 小红书图文…），一稿可裂变成多平台版本
- 每段可溯源回素材；配套审稿、改写、润色、拟标题等工具

## 🗺 在做什么 · 接下来

**飞轮的最后半环——「发布 → 数据 → 复盘 → 自进化」——正在做**：把发出去的内容表现接回来，让系统帮我看清"哪种写法真的让读者更有收获"，再反哺下一次选题。做完这半环，它才算真正闭环。
