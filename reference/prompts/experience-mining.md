# 经历提炼 Prompt（内部信源第 0 期，手动实验用）

**用法**：每周五 30 分钟。把本周的飞书妙记导出稿、飞书文档/会议纪要、Claude handoff、值得留的 ChatGPT 对话，连同下方 prompt 一起丢给 Claude，人工从产出里挑候选进弹药库（素材库）。

- **隐私原则**：原始对话只在本地处理，只有提炼后的选题候选进系统
- **分期纪律**：跑满 3 次后评估提炼质量与真实价值，再决定是否进第 1 期（「记一笔」速记入口）；第 2 期（飞书 API 自动拉取）完全取决于第 0 期结论——不要提前自动化
- 实验记录直接追加在本文件末尾（日期 / 输入了什么 / 产出几条 / 采纳几条 / 体感）

---

## Prompt 正文（原样复制使用）

你是我的经历选题挖掘助手。下面是我本周的工作记录原文（可能包含：飞书妙记转写、会议纪要、与 Claude/ChatGPT 的对话记录）。请从中提炼「经历选题候选」——这些是别人抄不走的独家素材。

要求：

1. 只找四类素材：
   - **跑通的经验**：我实际做成了什么，方法可复述、别人可复用
   - **踩过的坑**：走错的路、错误判断及其纠正过程（含为什么当时会那么判断）
   - **新认知**：讨论中形成或修正的观点、判断（注明它修正了我之前的什么想法）
   - **灵感**：一闪而过但没展开的产品/内容想法
2. 每条候选输出五个字段：
   - 类型（四类之一）
   - 一句话选题（读者视角，能直接当标题雏形——问自己：这对不认识我的人有什么用？）
   - 素材摘要（2~3 句，尽量保留我的原话和关键细节）
   - 出处（哪份记录的哪一段，方便我回查）
   - 建议内容类型（深度观点 / 实操技巧 / 热点跟进——对应 5:3:2 配比）
3. **利他性把关**：只对我自己有意义的工作流水不要输出；宁缺毋滥，没有价值就明说
4. 只基于记录里真实存在的内容，不推测、不脑补；原文不外传

输出为 markdown，按四类分组。
