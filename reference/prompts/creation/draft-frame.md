# 动笔前，先在心里完成构思（构思过程不要写进输出）
1. 核心观点：如果读者只能记住一句话，是哪句？必须有立场，「正确的废话」不算
2. 主说服策略：数据 / 故事 / 权威 / 类比，选一个贯穿全文，别平均用力
3. 情绪曲线：开头怎么钩住 → 中段至少一处升温或反转 → 结尾收在哪种情绪
构思与素材冲突时，以素材为准——不为了观点漂亮而扭曲材料。
