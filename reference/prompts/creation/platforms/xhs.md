把这份素材做成小红书「封面 + 内容卡」图文——这是给制图用的**卡片分解**（一屏一个信息点），不是一段长文。评判标准只有一条：读者每划一张卡，就多懂一件事或多拿一个能用的东西；讲不出增量的卡直接删。

# 整体
- 封面 1 张 + 内容卡最多 8 张，**含封面总共 ≤9 张（小红书图片硬上限）**；素材薄就少做几张，别灌水凑数
- 基调按「作者声音」，口语但克制；**不用浮夸营销词**（好用到哭 / 绝绝子 / 家人们 / 一惊一乍的刷屏感），不焦虑贩卖
- 【硬约束·不编造】「我试过/我用了/我踩过坑」这类第一人称，只有素材里真实存在才写；没有就用「一个常见现象是/有人发现」的第三方表述，禁止虚构亲历
- 数据、引语照素材写，用到就在句尾标 [素材N]

# 封面卡（决定点开率，最花心思的一张）
- 大字：≤12 字，一个钩子——痛点 / 反直觉结论 / 悬念（参照句式：「我替你踩过坑了」「先看这 5 个点」「别再手动 X 了」）
- 小字：≤22 字，补一句「谁该看」或「能带走什么」
- 大字别写成完整长标题，一张封面别塞多个主张

# 内容卡（一卡一个点）
每张卡两部分：
- 卡标题：一句话的判断或要点，读者只扫标题也能 get
- 正文：≤60 字，把这个点说清楚——给具体例子/数字/一句可直接抄走的话，不空泛
- 卡与卡有递进（痛点 → 为什么 → 怎么办）；第一张内容卡接住封面的钩子
- 需要引导往下看时，卡尾加「往下翻 ↓」（别每张都加）

# 正文文案（发布时放在图片下方那段）
- **整段 ≤1000 字符（小红书正文硬上限）**，超了先砍展开、保留钩子和行动建议
- 前 2 行：痛点或结论钩（信息流只露前 2 行，决定点开率）
- 中间 3-5 句：展开，有你的判断和体感
- 结尾：一句真实的行动建议或提问，不为骗互动硬凑
- 顺手埋 1-2 个读者会去搜的关键词

# 标签
3-5 个 #话题：1 个行业大词 + 2 个精准长尾 + 1 个热门，与主题强相关，不蹭无关热点
